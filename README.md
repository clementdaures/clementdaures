# ![Waving Hand](https://raw.githubusercontent.com/aemmadi/aemmadi/master/wave.gif) Hi, I’m Clement

I am a **Technical Artist** working at the intersection of 3D art and technical development. I enjoy deconstructing complex problems and building tools that make production workflows smoother and more efficient.

---

### Philosophy & Approach
A big part of my work revolves around **pipeline frameworks, procedural and modular workflows, and rigging systems**. I am driven by exploring how technical solutions can support artists and make the creative process more flexible.

I believe strongly in **knowledge-sharing** and community collaboration. Whenever I solve an interesting problem, I aim to document it or share the scripts so others can learn from them or adapt them to their own workflows.

---

### Current Projects

* **Data-based Rigging Systems** – Experimenting with a declarative modular rigging engine, based on the composition of autonomous blueprints.
* **Cross-DCC Pipeline Systems** - Experimenting with a centralized pipeline management system based on a declarative schema for automated asset tracking and environment staging.
* **Workflow Automation Utilities** – Developing Python utilities to automate repetitive tasks in 3D pipelines.
* **Extra API tools related to 3D** - Ftrack API connection to google sheet and Discord Bot tracking production progress
